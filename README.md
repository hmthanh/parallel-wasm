## Installation

1. Rustup and the Role of Rust Nightly

```
rustup toolchain install nightly
```

2. Overright nightly

```
rustup override set nightly
```

3. Run and build local

```
$ ./run.sh
```

4. Visit [http://localhost:8080](http://localhost:8080) or 127.0.0.1:8080
